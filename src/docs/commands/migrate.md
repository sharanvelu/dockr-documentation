# Dockr migrate

Runs the migration(`php artisan migrate`) command within the project container.

Usage :

```
dockr migrate [OPTIONS]
```

Example :

```
dockr migrate
dockr migrate rollback
dockr migrate refresh
```

::: tip
You can run all the migration commands in the same way
:::
